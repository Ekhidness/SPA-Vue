# Запуск проекта на локальной машине

## Требования:

- Node.js (версия 14 или выше)
- npm (входит в состав Node.js)
- Редактор кода (рекомендуется VS Code)

## Установка и запуск:

1. Клонирование репозитория

   ```
   git clone https://github.com/Ekhidness/SPA-Vue.git
   cd SPA-Vue
   ```

2. Установка зависимостей

   ```
   npm install --verbose
   ```

3. Запуск в режиме разработки
   ```
   npm run serve
   ```

Приложение будет доступно по адресу: http://localhost:8080

Важно: В браузере должен быть отключен режим HTTPS-Only, иначе запросы к API будут блокироваться.

4. Сборка для продакшена

```
 npm run build
```

Собранные файлы будут находиться в папке dist/.

Дополнительно:
Для настройки параметров сборки используйте файл vue.config.js в корне проекта.
Подробнее: https://cli.vuejs.org/config/
